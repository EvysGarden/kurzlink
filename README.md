# kurzlink
A static site generator for your shortlinks.

## Usage
```
cargo run -- --help
```

### `serve.sh`
This is a tiny script to quickly deploy a static server with python's http.server.  
It will simply generate the files and start a server on 0.0.0.0:8000.
